# ✨ 项目简介
一款轻量级浏览器脚本，帮助学生快速完成评教表单填写工作，提升评教效率。

# 🚀 功能特性
|功能|描述|
|---|---|
|⚡自动填写|一键完成所有评教项目|
|🛠️自定义配置|灵活设置评分和评语|
|🔒安全可靠|纯前端运行，不收集任何数据|
|📱多平台支持|兼容Chrome/Firefox/Edge等主流浏览器|

# 🛠️ 安装指南
1. 安装脚本管理器
   [脚本猫](https://scriptcat.org/zh-CN/)
2. 导入脚本
   
# 🎯 使用说明
进入评教界面，脚本自动执行

# ⚙️ 配置选项
评教界面可设置评教等级、评语

# ❗ 注意事项

⚠️ 使用前请确认：

学校评教系统是否允许自动化操作

脚本版本与评教系统兼容性

遵守学校相关规定
